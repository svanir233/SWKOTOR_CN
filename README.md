# SWKOTOR_CN

《Star Wars: Knights of the Old Republic》简体中文本地化项目。

当前仍在翻译中，暂未发布正式版本。

## 目录

| 路径 | 内容 |
| --- | --- |
| `translation/` | TLK 文本、术语表和翻译规则。 |
| `fonts/` | 源字体和 KOTOR 字体文件。 |
| `tools/` | 构建工具。 |

## 使用

构建工具见 `tools/README.md`。

## 参考资料

感谢以下公开资料对术语核对和格式研究的帮助。具体术语的条目级来源见 `translation/terms.csv`。

- [StarWars.com Databank](https://www.starwars.com/databank)：官方星战资料参考。
- [Wookieepedia](https://starwars.fandom.com/wiki/Main_Page) / [中文 Wookieepedia](https://starwars.fandom.com/zh/)：星战专名、设定和中文译名参考。
- [StrategyWiki: Star Wars: Knights of the Old Republic](https://strategywiki.org/wiki/Star_Wars:_Knights_of_the_Old_Republic)：KOTOR 任务、装备、技能和机制参考。
- [d20 SRD](https://www.d20srd.org/) / [DND 灰机 Wiki](https://dnd.huijiwiki.com/wiki/)：d20 规则术语参考。
- [NWN Wiki: TLK](https://nwn.wiki/display/NWN1/TLK) / [xoreos-docs: TalkTable_Format.pdf](https://github.com/xoreos/xoreos-docs/blob/master/specs/bioware/TalkTable_Format.pdf)：TLK 格式参考。
