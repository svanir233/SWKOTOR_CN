# SWKOTOR_CN

《Star Wars: Knights of the Old Republic》（SW:KOTOR）中文本地化资料项目。

## 文件

| 文件 | 说明 |
| --- | --- |
| `original_dialog.tlk.txt` | 原版 `dialog.tlk` 文本导出。只作为英文原文和 StrRef 基准。 |
| `translated_dialog.tlk.txt` | 中文译文稿。保持与原文相同的 StrRef 编号。 |
| `terms.csv` | 术语表。只收录 `star_wars` 和 `swrpg_rules` 两类术语。 |

## TLK 规则

- `original_dialog.tlk.txt` 不写译文。
- `translated_dialog.tlk.txt` 保留原 StrRef。
- 本项目不使用机器翻译；`translated_dialog.tlk.txt` 只能写人工译文。
- `<CUSTOM0>`、`<CUSTOM1>` 等运行时占位变量必须逐字保留，编号和顺序不得改动。
- 翻译每个 StrRef 前必须先查 `terms.csv`；原文命中 `source_term` 时，先采用表内 `zh_cn`。
- 只有普通词、子词误命中或 `note` 明示需按上下文判断时，译文才能偏离 `terms.csv`。
- 同一个英文词在不同 StrRef 中含义不同时，按原文上下文翻译。
- 原版导出为 `48560` 条字符串；最后 StrRef 是 `49264`，原版 StrRef 不连续，不能用最后编号判断总条目数。

## Terms 规则

`terms.csv` 只收录需要证据链的术语。

| 字段 | 规则 |
| --- | --- |
| `domain` | 只能是 `star_wars` 或 `swrpg_rules`。 |
| `source_term` | 英文原词。 |
| `zh_cn` | 当前简体中文译名。 |
| `count` | 在原文中的出现次数或整理时统计值。 |
| `status` | 只能是 `confirmed` 或 `review`。 |
| `en_reference` | 支撑英文概念的英文来源。 |
| `zh_reference` | 支撑中文译名的中文来源。 |
| `note` | 只写必要说明、版本差异或 KOTOR 语境。 |

不进入 `terms.csv`：

- UI 文案
- 普通装备词
- 普通动作词
- 普通形容词
- 任何必须依赖具体句子判断的词

## Reference 规则

`en_reference` 和 `zh_reference` 必须可复核。

`en_reference` 有效格式：

- 英文网页：具体条目页 URL。
- 英文书籍：`book=书名|出版社|ISBN|页码或章节`
- KOTOR 原文语境：`tlk=StrRef 1234`

`zh_reference` 有效格式：

- 中文网页：具体条目页 URL。
- 中文书籍：`book=书名|出版社|ISBN|页码或章节`

无效格式：

- `sw_databank`
- `official_zh`
- `wotc_official_zh`
- `d20_core`
- `kotor_tlk`
- 任何不能直接打开或核查的粗标签

`en_reference` 和 `tlk=StrRef ...` 只能证明概念或游戏上下文，不能证明中文译名。

## Status 规则

`confirmed` 必须同时满足以下条件：

- `en_reference` 有具体英文网页 URL、完整英文书目信息或 `tlk=StrRef ...`，支撑英文概念或游戏语境。
- `zh_reference` 有具体中文网页 URL 或完整中文书目信息，支撑中文译名。

`review` 用于以下情况：

- 只有 `en_reference`，没有 `zh_reference`。
- 只有 `zh_reference`，没有 `en_reference`。
- 没有可复核中文来源。
- 只有 KOTOR 原文语境。
- 中文译名是项目暂定。

## Star Wars 术语

`domain=star_wars` 收录 Star Wars 宇宙专名，包括人物、地点、组织、物种、技术、武器、飞船、星球和 KOTOR 专名。

来源规则：

1. 有官方中文译名：使用官方中文；`zh_reference` 写具体官方 URL 或完整书目信息。
2. 无官方中文译名：使用中文 Wookieepedia / 星球大战百科全书具体条目页，并写入 `zh_reference`。
3. StarWars.com Databank 和英文 Wookieepedia 只能写入 `en_reference`，不能单独确认中文译名。
4. 没有中文来源：`status=review`，`zh_reference` 留空。

KOTOR 专名如果能在 Star Wars 来源中查到，仍归入 `star_wars`，不单设 `kotor` 域。

## SWRPG 规则术语

`domain=swrpg_rules` 收录 KOTOR/KOTOR2 的规则术语，包括属性、检定、豁免、专长、加值、伤害类型、职业、技能和状态。

规则依据：

- WotC 的 `Star Wars Roleplaying Game Core Rulebook` 出版于 2000 年，`Star Wars Roleplaying Game Revised Core Rulebook` 出版于 2002 年；这条规则基于 D&D 3E 使用的 d20 System。
- KOTOR 的规则系统基于 WotC `Star Wars Roleplaying Game`；KOTOR2 沿用并改造同一 d20 规则。
- 原规则书年代久远且已绝版，稳定可复核的官方电子版不能作为默认来源；不使用来源不明的扫描版当来源。

依据来源：

- `https://starwars.fandom.com/wiki/Star_Wars_Roleplaying_Game`
- `https://en.wikipedia.org/wiki/Star_Wars%3A_Knights_of_the_Old_Republic`
- `https://en.wikipedia.org/wiki/Star_Wars_Knights_of_the_Old_Republic_II%3A_The_Sith_Lords`

来源规则：

1. 有官方中文译名：使用官方中文；`zh_reference` 写具体官方 URL 或完整书目信息。
2. 能核到原规则书或游戏手册时，按语言写入 `zh_reference` 或 `en_reference`，格式为 `book=书名|出版社|ISBN|页码或章节`。
3. 核不到原规则书页码时，可以用相近的 WotC d20、D&D 3E/3.5、d20 SRD 或 DND 灰机 Wiki 具体页作为机制辅助来源；中文页写 `zh_reference`，英文页写 `en_reference`。
4. Star Wars 专名、阵营、原力、种族、地点和组织按 `star_wars` 来源规则处理。
5. 兼容来源只能支撑概念，不单独把译名标成 `confirmed`。
6. 没有可复核中文来源或完整书目信息：`status=review`，`zh_reference` 留空，可只写 `en_reference`。

## 冲突规则

- 没有 `en_reference` 或 `zh_reference` 的说法不作为依据。
- 来源冲突时，采用更贴合 KOTOR 语境且可复核的译名。
- 冲突原因写入 `note`。
